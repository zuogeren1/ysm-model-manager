# 🎨 v1.6.0 - UI 系统全面优化

**发布日期：** 2026-06-15

---

## ✨ 新功能与改进

### 🌟 响应式布局支持
应用现在可以自动适配不同屏幕尺寸：
- **大屏幕（>1200px）：** 完整三栏布局（侧栏 + 主内容 + 预览面板）
- **中等屏幕（900-1200px）：** 两栏布局（隐藏预览面板，聚焦核心功能）
- **小屏幕（<900px）：** 单栏布局（仅显示主内容区）
- 窗口大小调整时平滑过渡动画

### ♿ 无障碍支持升级
- 键盘导航时显示清晰的焦点环（按 Tab 键体验）
- 鼠标点击不显示焦点，避免视觉干扰
- 符合 WCAG 2.1 无障碍标准，提升可访问性

### 🎨 统一按钮样式系统
- 新增 `.btn-base` 基类，提供一致的按钮外观
- 三种尺寸：小（工具栏）、中（默认）、大（主要操作）
- 语义化变体：主要、危险、高亮
- 所有现有按钮保持兼容，无需修改

### 🏗️ Z-Index 层级优化
- 统一管理浮层元素的显示层级
- 修复可能的层级冲突问题
- Toast 通知、右键菜单、全屏预览等现在正确叠加显示

---

## 🔧 技术改进

### 性能优化
- 移除不必要的 `will-change` 属性，减少内存占用
- 优化 CSS 过渡动画，指定具体属性而非 `all`
- 提升大量卡片渲染时的流畅度

### 代码质量
- CSS 变量语义化重命名（更易理解）
- 移除 4 处 `!important` 滥用，提高可维护性
- 添加完整的 CSS 变量文档注释

### 向后兼容
所有改动都保持了向后兼容：
- 旧按钮类仍然可用（`.tb-btn`, `.dlg-btn` 等）
- CSS 变量通过别名映射，现有样式不受影响
- 无破坏性改动，升级无忧
