# v1.3.1 — CSS 工程化 + Shadow DOM 样式隔离修复

> 发布日期: 2026-06-09

## 🎯 核心改动

### 🎨 CSS 搬迁移至 Shadow DOM（修复创意工坊/诊断页/设置页样式丢失）

之前将 CSS 从 `components.css`（全局）迁移到 `content-css.js`（Shadow DOM）时，大量 workshop 相关的 CSS 类被遗漏，导致全局 CSS **无法穿透 Shadow DOM**，创意工坊、诊断页、设置页出现不同程度的样式缺失（按钮无样式、搜索框无边框、数据源徽标不显示等）。已全部修复。

### 🐞 Bug 修复

- **模型行 `.ws-right` 列布局冲突** — 行右分组改名为 `.ws-row-right`
- **设置页链接提示从不显示** — 改用 `display = "block"`
- **链接提示文字颜色硬编码** — 改为 `color:var(--muted,#6c7086)`
- **两个 `.hdr-btn` 规则重复** — 保留一份

## 📝 说明

本次发布不包含 3D 模型预览的改进（v1.3.0 草稿中的 3D 渲染引擎翻修将在后续版本正式发布）。
