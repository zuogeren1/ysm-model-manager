# v1.3.6 — 更新更稳 + 更新日志直接看

> 发布日期: 2026-06-10

## ✨ 新功能

### 📬 启动时静默检查更新

启动后自动检查新版本，发现更新就在右下角弹个小提示，点击后一键下载安装，不再弹出黑框。

### 📋 更新弹窗直接看发布说明

发现新版本后弹窗直接显示更新日志，不用再跳转浏览器。如果你跳版本更新（如 v1.3.3 → v1.3.6），弹窗会**聚合中间所有版本的更新内容**。

## 🔧 改进

- **更新器重写** — 以前更新会弹 cmd 黑框（`updater.bat`），现在后台静默完成，无黑框、可回滚
- **弹窗优化** — 更新日志区域带边框背景，内容多时可滚动，适配三套主题
