# v1.3.6 — 更新器重构 + 静默检测 + 更新日志聚合

> 发布日期: 2026-06-10

## 🎯 核心改动

### 🔄 更新器重构（LytVPK 风格）

旧方案使用 `updater.bat`（cmd 黑框 + xcopy），新方案改为 Go 端直接操作文件：

| 步骤 | 旧（batch） | 新（Go） |
|------|-------------|----------|
| 解压 | `Expand-Archive` PowerShell | `archive/zip` 原生 |
| 替换 | `updater.bat` → `xcopy` | `os.Rename(exe → exe.old)` → `os.Rename(exe.new → exe)` |
| 回滚 | 无 | 替换失败自动 `os.Rename(oldPath → exe)` 恢复 |
| 清理 | 无 | 启动时自动删 `exe.old` |
| 重启 | `window.close()` | Go 端 `exec.Command(self).Start()` + `runtime.Quit()` |
| 进度 | 无 | 支持 `EventsOn("update_progress")` |

### 📦 静默更新检查

- 应用启动后**异步**检查 GitHub 新版本，不阻塞界面
- **6 小时限频**（`localStorage` 记录上次检查时间）
- 发现新版本 → 右下角可点击的 toast 通知
- 点击 toast → 弹出更新确认对话框
- 检查失败静默忽略

### 📋 更新日志聚合

- 从 `/releases/latest`（仅最新）改为 `/releases?per_page=10`（最近 10 个发布）
- 遍历所有比当前版本新的 release，聚合发布说明
- 更新弹窗直接显示格式化的更新日志（带边框 + 主题色背景）

### 🎨 更新弹窗 UI

- 自定义 modal，不再使用通用 `modalConfirm`
- 更新日志区域：`border: 1px solid var(--bd)` + `background: var(--surf)`
- 内容过长自动滚动（`max-height: 40vh; overflow-y: auto`）
- 自适应三套主题（赛博霓虹 / 温暖木纹 / 极简深邃）
- 对话框宽度 480px，比默认更紧凑

### 📄 署名与文档

- 设置页「关于」新增灵感来源署名块
- README 新增灵感来源表格（LytVPK / YSMViewer / YSMParser / MO2）
- README 底部添加 Apache-2.0 许可证声明

### ⚠️ 已知问题

| 问题 | 影响 | 状态 |
|------|------|------|
| 双手臂模型（如 沐白）手臂骨骼/偏移未正确合并 | 从 ZIP 解压后 `main.json` + `arm.json` 的手臂立方体重复渲染 | 待骨骼合并算法修正 |
| 立方体数量 > 2000 的模型 3D 预览卡顿 | 全分辨率纹理渲染 + 无 LOD，高面数模型帧率低 | 待纹理降采样或 LOD 优化 |
