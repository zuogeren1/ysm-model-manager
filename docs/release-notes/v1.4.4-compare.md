## 关键修复

### health 公式重算

```
旧: 100 - min(重复×5,30) - min(禁用×3,15) + min(数量/10×2,10) → 全禁用95分
新: 100 - min(禁用占比×40,40) - min(重复×5,55) → 全禁用60分 ✓
```

### 自动搜索

```
修复前: filepath.Join("C:", "PCL2", ".minecraft") → C:PCL2\.minecraft（相对路径，找不到）
修复后: "C:\\PCL2\\.minecraft" → 正确绝对路径 ✓
```

### YSM 标签解析

```
修复前: <name>TestModel</name> → value = "TestModel</name>"
修复后: stripClosingTag → value = "TestModel" ✓
```

### Go 测试框架

```
之前: 0 个测试
现在: 20 个测试（ysm 14 + sync 6），go test ./go/... 一键运行
```
