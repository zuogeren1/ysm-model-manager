## ✨ 新功能

### 创作者卡片系统

| 功能                | 说明                                                           |
| ------------------- | -------------------------------------------------------------- |
| 金银铜牌排名        | 根据仓库内模型数降序排列，前三名分别获得🥇🥈🥉，第四及以后获💠 |
| 卡片旋转动画        | 悬停时卡片 3D 旋转 + 呼吸缩放（非前三名淡蓝色静态边框）        |
| 平台徽章            | 卡片左下角显示平台来源（Booth/NicoNico/VRoid 等）              |
| 身份标签 + 描述标签 | `role` 字段（个人势/社团/官方）+ `desc` 拆标签显示             |
| 本地模型关联        | 点击详情浮层查看该创作者在本地仓库中的模型列表                 |
| 收藏创作者 ⭐       | Tab 状态持久化，收藏创作者置顶显示                             |
| 搜索增强            | 同时搜索创作者名和简介                                         |
| 创作者卡片拖拽排序  | 编辑模式支持 ↑↓ 排序按钮 + ⠿手柄拖拽，目标卡片让出空间动画     |

### 详情浮层

| 特性         | 说明                                         |
| ------------ | -------------------------------------------- |
| 果冻弹入动画 | 打开时弹性动画，圆角 16px                    |
| 信息分层     | 用户视角文案 + 描述卡片背景 + ⭐置右         |
| 本地模型卡片 | 浮层内显示该创作者在仓库中的模型，点击可跳转 |
| 注入全局 CSS | Shadow DOM 不穿透主文档，CSS 独立注入        |

### 分类标签系统

| 特性          | 说明                                                  |
| ------------- | ----------------------------------------------------- |
| `tag` 字段    | 每条创作者记录新增 `tag` 数组（game/vup/原创/官方等） |
| 标签筛选栏    | 顶部标签行筛选，默认选中 `game` 标签徽章              |
| type 多选下拉 | 资源类型（YSM/MMD/VRC 等）多选筛选                    |

### 头像提取 & 缓存

| 功能                | 说明                                                   |
| ------------------- | ------------------------------------------------------ |
| 从本地 YSM 提取头像 | `DebugExtractCreatorAvatar` 解析模型的 `model/avatar/` |
| 批量缓存            | 解析完自动缓存到头像目录，下次直接从文件读取           |
| 仅解析 .ysm 文件    | 跳过 zip/7z，避免误解析大文件（`2ee0405`）             |
| 头像调试入口        | 点击头像触发 debug 模式（`__debugAvatar` 全局暴露）    |
