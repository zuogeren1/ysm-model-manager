# v1.3.4 — 全局 CSS 加载修复（对话框/批量重命名/滚动条恢复）

> 发布日期: 2026-06-10

## 🎯 核心改动

### 🎨 修复全局 CSS 未加载导致 UI 样式丢失

`frontend/css/components.css`、`layout.css`、`scrollbar.css` 通过 `style.css` → `@import` 链存在，但 `style.css` **从未被任何文件引用**，导致构建产物中全部全局样式缺失。

**影响范围**：所有依赖全局 CSS 的 UI 元素——重命名弹窗、批量重命名、模态框、对话框、布局、滚动条等。

**修复**：在 `index.html` 中直接链接 CSS 文件：

```html
<link rel="stylesheet" href="css/variables.css" />
<link rel="stylesheet" href="css/layout.css" />
<link rel="stylesheet" href="css/components.css" />
<link rel="stylesheet" href="css/scrollbar.css" />
```

构建 CSS 体积：**1.32 KiB → 10.21 KiB**（模块数 76 → 79）。

### 📁 影响文件

| 文件                  | 改动                                                      |
| --------------------- | --------------------------------------------------------- |
| `frontend/index.html` | 新增 `components.css`、`layout.css`、`scrollbar.css` 链接 |
