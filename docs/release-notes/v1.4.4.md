# v1.4.4 — 健康度公式修复 + 自动搜索修复 + Go 测试框架

> 发布日期: 2026-06-11

## 🔧 Bug 修复

### 健康度评分重写

- 旧公式：`100 - min(重复×5, 30) - min(禁用×3, 15) + min(数量/10×2, 10)`
  - 全禁用的仓库仍然显示 95 分，明显不准确
- 新公式：`100 - min(禁用占比×40, 40) - min(重复×5, 55)`
  - 全禁用仓库 → 60 分，全健康无重复 → 100 分
  - 重复比禁用更扣分（重复更占空间）

### .minecraft 自动搜索修复

- **路径 Bug**：`filepath.Join("C:", "PCL2", ".minecraft")` 在 Go Windows 上产生的是相对路径 `C:PCL2\.minecraft`，而非 `C:\PCL2\.minecraft`，导致自动搜索找不到 PCL2 等启动器的 .minecraft 目录
- **特征检测**：新增 `mods`、`config` 目录作为 .minecraft 识别标记
- **父目录扫描**：新增第 4 段扫描——遍历已知启动器目录，检查其中是否包含 `.minecraft` 子文件夹（不要求特征文件），覆盖非标路径

- **parseInt 不支持负数/空格**：`parseInt("-5")` 返回 0 而非 -5，`parseInt("3.14")` 返回 3 而非 0
