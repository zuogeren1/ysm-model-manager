# v1.5.2 — 版本列表重构·多类型同步·推拉体系统一

> 发布日期: 2026-06-13

---

## ✨ 新特性

### 整合包版本列表重构

- **左右数据同步打通**：点击 sync-manager 类型标签（💎 YSM/🎭 MMD/🥽 VRC/🎨 材质包/☀️ 光影包/⚙️ 蓝图）→ 左侧 sidebar 卡片数字实时刷新，不再依赖完整重载
- **类型联动**：默认初始选中 YSM，不跟随仓库页 subtab 选择，避免混淆
- **卡片精简**：纯数字标签（`172` `30` `3`），无文字无图标，`📦` 图标下移到数字区
- **按数字排序**：有 mod 的包排前面，无 mod 排最后，同类按已同步数降序
- **切换类型后重排**：切换类型标签后卡片位置按新数据排序

### 勾选 + 同步所选

- 每个整合包卡片左侧复选框 `☐`
- 顶部 `☐ 全选` + `⬇️ 同步所选` 按钮，只对勾选的整合包安装缺失文件
- 无勾选时提示"请先勾选要同步的整合包"

### 状态标签行替代原筛选栏

- 移除搜索框和状态下拉菜单，改为 `📊 全部` `✅ 已同步` `⬇️ 缺失` `⛔ 已禁用` `📤 可拉取` 标签行
- 数字集成到按钮文本（推送全部 (30) / 拉取全部 (4)）
- 列表每行 `·` 灰色圆点统一，状态由筛选标签指示

### MMD/YSM 文件夹推送

- **MMD 模型**（`.pmx`/`.pmd`）推送整个文件夹（含 `spa/` `textures/` `toon/` 子目录及所有纹理文件）
- **YSM 模型**（`ysm.json`）在子目录中时同样整文件夹推送
- **VRM 文件**是自包含格式，单文件推送
- 新增 `InstallDir()` 递归复制子目录 + 按类型过滤文件（MMD 排除 `.vrm`）

### 扩展名全面支持

- `.json` 加入全部扫描点（仅 `ysm.json`，排除动作/动画文件）
- `.vrm` 加入 🥽 VRC 类型支持
- 文件树按类型过滤扩展名，共享仓库不混入其他类型文件
- 导入按扩展名自动路由到对应仓库目录

### ⛔ 已禁用状态

- 禁用文件（`.ban`/`.disabled`）在 sync-manager 中标记 `⛔`，可筛选查看
- `GetInstanceSyncStatus` 正确识别禁用文件
- **注意：回收站 `os.Remove` 的安全机制 **

### 清空操作安全增强

- `CountInstanceResources` 先计数再删除，0 文件时跳过
- 确认对话框显示扫描到的文件数
- `countInstanceDir`/`clearInstanceDir` 改为 `filepath.WalkDir` 递归遍历子目录
- 删除文件后清理空目录

## 🔧 技术改进

### 新增 Go Binding

| Binding                          | 文件                        | 说明                                       |
| -------------------------------- | --------------------------- | ------------------------------------------ |
| `CountInstanceResources`         | `app_install.go`            | 统计整合包中可清空的文件数                 |
| `PullSingleResourceFromInstance` | `app_install.go`            | 单文件拉取（从整合包到全局仓库）           |
| `ClearInstanceResources`         | `app_install.go`            | 清空整合包所有资源类型文件（跨子目录递归） |
| `installer.InstallDir`           | `go/installer/installer.go` | 整目录安装（递归 + 类型过滤）              |

### 事件系统改进

- 卡片点击/右键从 `hdr.onclick`/`oncontextmenu` 改为 `#vg` 容器级事件委托
- 新增 `sidebar:rtype-changed` 事件，sync-manager 初始化时也 emit
- 废弃 `sidebar-import-all` 按钮，替换为全选 checkbox + 同步所选按钮

### 调试日志

- `InstallResourceToInstance` 全链路日志（rtype / srcPath / globalRoot / dstDir / 条件分支）
- `InstallDir` 日志（目录创建 / 文件处理 / 子目录递归）
- 保留调试日志，供后续排查
