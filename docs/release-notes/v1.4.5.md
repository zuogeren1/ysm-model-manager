# v1.4.5 — 设置页卡片化 + 主题色按钮修复 + 按钮与字号统一

> 发布日期: 2026-06-11

## 🎨 UI/UX 优化

### 设置页全面卡片化

- 「存储策略 & 网络」改为双栏卡片并排，链接模式从 radio 改为下拉菜单
- 「字体与布局」改为三栏卡片并排（基准字号 / 创作者字体 / 卡片密度）
- 新增「ℹ️ 关于」独立标签页，卡片式展示项目介绍、技术栈、资源链接、快速上手、灵感来源
- 下载镜像源切换时显示详细提示卡片（直连 / jsDelivr / GitHub API 各自说明）

### 字体与主题系统统一

- 全面修复 `adoptedStyleSheets` 中 CSS 变量不继承的问题（`var(--accent)` 硬编码 + `:host-context()` 主题适配）
- 补全 `.btn` 按钮系列 CSS 类（`.btn`、`.btn.accent`、`.btn.danger`），所有按钮统一主题色
- 清除各处硬编码 `font-size`，改用 CSS 变量体系
- 按钮和标题字号调高一级（提示 9→10px，标题 12→13px）
- 仓库元老页面各分区标题字号调高（📊 健康度、📅 月度活动、🎲 今日推荐等）

### 细节打磨

- 创作者频道外链/内嵌切换改为 pill 式开关，移至 tab 栏右侧
- 返回/浏览器打开按钮统一样式（`ws-back`、`cr-back-btn` 等）
- 🗑️ 清空按钮 padding 修复，与 ✕ 按钮一致
- 暂无文件字号与已导入计数对齐
- 文件名预览行 `overflow:hidden` 修复长文件名撑爆布局

## 🔧 功能改进

### 存储策略

- 切换链接模式后**自动执行重新链接**，不再弹确认框
- 新增「🔄 重新应用链接」按钮，可手动触发
- 安装按钮文字简化：`⬇️ 安装缺失 (N)` → `安装 (N)`，`⬇️ 安装` → `安装`

### 拖拽导入

- 修复从其他页面拖拽文件导航到错误界面（`"downloads"` → `"repository"` + 自动切换到导入标签）
- 新增 `repo:switch-tab` 全局事件机制

### 关于页面

- 重新组织为卡片布局：定位介绍、技术栈、资源链接、快速上手、灵感来源
- 新增「快速上手」指引（5 步），降低新用户门槛
- 灵感来源增加详细关系描述，不再只是链接列表
- 同步更新 README 开篇描述
