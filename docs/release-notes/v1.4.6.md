# v1.4.6 — 自动更新修复 + 渐变头像边框 + 呼吸灯动画

> 发布日期: 2026-06-11

## 🐛 Bug 修复

### 自动更新选错 exe

- **根因**：`updater.go` 中匹配 ZIP 内第一个 `.exe` 文件，可能选到 `ysm-cli.exe` 而非 `YSM-Model-Manager.exe`
- **修复**：精确匹配 `YSM-Model-Manager.exe`（`strings.EqualFold`），排除其他 exe
- **影响**：修复后自动更新不会再错误地替换为 CLI 工具

## ✨ 新特性

### 创作者头像渐变边框

- 作品数越高边框越亮，从银灰到亮金平滑过渡
- `conic-gradient` 环形渐变 + `brightness` 呼吸灯动画（3s 周期）
- hover 时包裹层旋转 + 放大反馈

### 呼吸灯动画系统

- **健康度环**：微弱呼吸（4s 周期），让统计数据更有生命力
- **下载进度条**：下载中亮度脉动，提示操作进行中
