## 🎨 改进

### MMD 变体折叠显示

同一 MMD 文件夹下的多个 `.pmx` 变体（如"二套衣服.pmx""獸爪.pmx""婚纱.pmx"）在预览面板中聚合为可折叠卡片，默认展开，提供"同步全部变体"按钮。

影响范围：仅 `mmd-skin` 类型，YSM/其他资源类型不受影响。

### 事件命名统一

| 旧名                    | 新名                    |
| ----------------------- | ----------------------- |
| `sync:download-missing` | `sync:download:missing` |
| `sync:toggle-status`    | `sync:toggle:status`    |

新增 `docs/event-catalog.md` 事件目录文档。

### 体验优化

- context-menu 四边溢出检测 + 跳变消除（先测量再设置位置）
- Toast 最多同时 5 个，超出移除最早的
- 拖拽文件上限 10MB → 100MB
- 折叠图标 ▾ 展开/收起动画（grid-template-rows + rotate）
- rename.js 头部读取失败时显示提示而非静默
