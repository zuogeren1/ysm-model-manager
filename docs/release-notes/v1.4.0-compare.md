## 实现细节

### 筛选连接
`runFilter()` → `SearchModels` Go 后端 → `filter:results` → `_filterPaths` Set → `buildTree` 过滤 → 树刷新。

### 纹理尺寸提取
`ExtractYsmSummary` 解析 ysm.json 时，遍历 `files.player.model` 路径，打开对应 geometry JSON，解析 `minecraft:geometry[0].description.texture_width/height`。

### 健康度评分
100 分 - 重复扣分(每个×5, 上限30) - 禁用扣分(每个×3, 上限15) + 模型数加分(每10个+2, 上限10)。
