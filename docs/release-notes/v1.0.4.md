## v1.0.4 — 创意工坊 GitHub 仓库浏览 + 自动索引

### ✨ 新功能

- **🌐 GitHub 仓库在线浏览**：创意工坊中点击「📦 浏览」直接读取远程仓库的 index.json，在应用内浏览模型列表，无需打开浏览器
- **📥 一键下载模型**：列表中的每个模型可单独 ⬇️ 下载到本地仓库
- **⬇️ 批量下载缺失**：自动比对本地已有文件（hash + 文件名双策略），一键下载所有缺失模型，每 10 秒一个防 GitHub API 限流
- **♻️ 双回退下载源**：raw.githubusercontent.com → api.github.com 自动回退，解决国内 CDN 不稳定问题
- **🔐 index.json hash 字段**：生成索引时自动计算 SHA256，下载时按 hash 去重，改名不变内容不重复下载
- **🤖 GitHub Action 自动索引**：GenerateRepoIndex 自动生成 `.github/workflows/generate-index.yml`，推送模型文件后自动更新 index.json
- **📇 生成 GitHub 索引增强**：同时输出 index.json + workflow 文件，文案更清晰

### 🐛 修复

- 右键菜单被 document contextmenu handler 立即隐藏（`e.stopPropagation()`）
- GitHub API 返回的中文模型名乱码（base64 → Uint8Array → TextDecoder）
- esbuild 构建报错 `Parse error @:1626`（`.map()` 中缺失 `return`）
- 错误状态没有返回按钮，用户被困在错误页面

### 🎨 体验优化

- **倒计时进度条**：连接时从 50% 开始每秒减 10%，条纹动画 + 旋转沙漏，不再干等
- **API 来源标签**：标题栏显示当前用的是 raw 还是 API
- **默认仅显示缺失**：列表默认只显示未下载的模型，可切换「显示全部」
- **右键查看索引信息**：右键模型行查看文件名/路径/hash/大小
- **文件大小限制**：>4MB 弹确认 >10MB 拒绝
- **下载进度条**：Go 端通过 runtime.EventsEmit 实时推送百分比
- **超时 10s → 5s**，两个源合最多等 10 秒
- **错误提示优化**：无 index.json 时显示引导说明 + 返回按钮
- **Toast 支持换行**：多行消息用 `\n` 分段显示
- **搜索框**：实时按名称过滤模型

### 🔧 技术改进

- `DownloadFromGitHub` 新增 `progressReader` 流式进度推送
- `GenerateRepoIndex` 增加 hash 字段 + 自动生成 GitHub Action workflow
- 前端 `showRepoModels` 按 hash 比对本地文件，支持多源渲染
