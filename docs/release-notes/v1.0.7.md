# v1.0.7 — 模型预览增强 + 日志筛选 + UX 清理

> 发布日期: 2026-06-07

## 🎯 核心改动

### 🖼️ 模型预览三级回退

- 同目录散图：自动查找 `.png/.jpg/.jpeg/.webp/.gif`，并行尝试，取最先成功的
- zip/7z 解压纹理：压缩包内提取第一张 `.png` 作为缩略图
- 无图时显示文字卡片（已有）

### 📦 整合包封面

- 点击文件夹自动检测 `ysm-pack.json` + `ysm-pack.png`
- 优先读取 `lang.zh_cn` 中文名/描述
- 显示整合包名称 + 描述 + 封面图

### 📋 导入日志筛选

- 整合包预览面板日志 + 诊断页日志，双向支持：
  - 状态筛选（全部/成功/失败/跳过）
  - 搜索框（按模型名实时过滤）
- 安装成功操作现在也记录日志（之前只记录失败）

### 🧹 UX 清理

- 创意工坊删除死按钮（📥📤 导入导出）
- 编辑创作者删除导入导出按钮
- 所有配置文件页面标注数据来源：
  - 设置页 → `ysm_config.json`
  - 创意工坊 → `workshop_sites.json` / `workshop_creators.json`
- 日志列表增加滚动条

### 🐛 修复

- 安装成功操作现在正确记录日志
- 日志列表大数量时添加滚动条（max-height: 300px）
