# v1.5.1 — 解压模型预览 & 稳定性修复

> 发布日期: 2026-06-13

---

## ✨ 新特性

### 解压 YSM 模型预览

- **支持直接点击 `ysm.json`（解压后的 YSM 模型目录）预览**：
  - 右侧摘要面板正确显示模型名称、作者、许可证、纹理/模型数量
  - 2D 骨架预览自动查找 `models/main.json`（优先 "main" key）
  - 纹理缩略图从 `textures/` 子目录自动提取
  - 递归搜索子目录中的 geometry JSON（排除 `animations/` / `controller/` / `avatar/` 目录）
- `ExtractYsmSummary` 新增裸 `ysm.json` 解析支持（`spec`/`metadata`/`files` 完整结构）
- `AnalyzeBedrockModel` 新增 `.json` 分支：解析 ysm.json 元数据 → 定位 model 文件 → `textures/` 纹理

### 搜索增强

- 递归搜索 geometry 时排除 `animations/` / `controller/` / `avatar/` 目录，避免动作文件被误识别
- `files.player.model` 支持三种格式：对象 `{"main":"..."}`、数组 `["..."]`、单字符串
- 对象格式时优先选择 key 为 `"main"` 的 model 文件

## 🔧 技术改进

### 解压 YSM 模型数据流

```
点击 ysm.json
  → WASM parseYsmJsonDirect 识别 ysm.json 格式
  → 返回占位 geometry（含 _ysmMeta）
  → Go AnalyzeBedrockModel(.json 分支):
      ① 解析 ysm.json → files.player.model = {"main":"models/main.json"}
      ② 优先查找 key="main" → models/main.json
      ③ parseBedrockGeometry → 33 bones ✅
      ④ 纹理: 搜索 textures/*.png
  → 2D 骨架预览 + 纹理缩略图
```
