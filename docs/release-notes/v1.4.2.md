# v1.4.2 — 按钮绑定修复 + 创作者数据库扩充

> 发布日期: 2026-06-11

## 🔧 Bug 修复

### 整合包安装按钮全面修复

- **单个「⬇️ 安装」按钮**：修复因 Shadow DOM 隔离导致的事件绑定失效。按钮渲染在右侧面板（`app-content` Shadow DOM），但绑定代码在侧栏（`app-sidebar` Shadow DOM），`querySelector` 跨不过隔离边界。现改为在渲染位置直接绑定。
- **「一键安装模型」按钮**：修复事件名不匹配导致按钮永久卡在「⏳ 导入中」的问题。监听事件从 `queue:status` 改为 `sync:download-complete`。
- 清理 `actions.js` 中遗留的死代码（已迁移的 `btn-install-one` 绑定）。

### 仓库元老页面修复

- 修复 `loadOldestModel` 未 `export` 导致动态 `import()` 拿不到函数的问题。
- 进入「仓库元老」页面时隐藏诊断页左菜单栏，避免界面混乱。
