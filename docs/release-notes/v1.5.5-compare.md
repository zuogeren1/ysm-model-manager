# v1.5.5 — 全量前端审计修复 🔍🔧

## 变更内容

### 全量前端审计

本次发布是一次全面的前端代码审计修复，覆盖了 `app-sidebar`、`app-tree`、`app-preview`、`app-sync-manager`、`app-nav`、`app-toast`、`context-menu`、`app-tree-styles`、`core/` 全部 8 个文件、`dialogs/` 全部 3 个文件，以及 Go 端扩展名统一。

### 影响范围

- **重大**：多处内存泄漏修复、XSS 防护加固、dead code 清理。Go 端 `sync.go` 的 `SyncToggleStatus` 扩展名过滤 Bug 修复（遗漏 `.json/.pmx/.pmd` 等类型）。
- **轻微**：MMD 变体折叠显示、事件命名统一（向后兼容通过 `context-menus.js` 桥接）。
- **无**：API/接口变更。
