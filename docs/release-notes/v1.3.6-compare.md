# v1.3.6 — 开发者对比（技术细节）

> 此文件记录实现细节，供开发者参考，不展示给用户。

## 🔄 更新器实现对比

| 步骤 | 旧（batch） | 新（Go） |
|------|-------------|----------|
| 解压 | `Expand-Archive` PowerShell | `archive/zip` 原生 |
| 替换 | `updater.bat` → `xcopy` | `os.Rename(exe → exe.old)` → `os.Rename(exe.new → exe)` |
| 回滚 | 无 | 替换失败自动 `os.Rename(oldPath → exe)` 恢复 |
| 清理 | 无 | 启动时自动删 `exe.old` |
| 重启 | `window.close()` | Go 端 `exec.Command(self).Start()` + `runtime.Quit()` |
| 进度 | 无 | 支持 `EventsOn("update_progress")` |

## 📁 影响文件

| 文件 | 改动 |
|------|------|
| `go/updater/update.go` | 重写 `Check`（多版本聚合）、`InstallUpdate`（os.Rename 替换）、`CleanupOldVersion` |

| `frontend/js/components/app-content/tpl.js` | 设置页新增灵感来源署名 |
| `README.md` | 灵感来源表格 + 许可证声明 |
