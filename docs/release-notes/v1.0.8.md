# v1.0.8 — 2D 模型预览 + YSM 格式识别

> 发布日期: 2026-06-07

## 🎯 核心改动

### 🖼️ 2D 模型骨骼渲染

- 新增 Canvas 2D 骨骼线条图渲染（前视图 + 俯视图小窗），展示模型骨骼结构
- `renderModel2D()` 自动居中缩放，支持纹理贴图加载
- 颜色可见度修复：填充色 rgba 从 0.15 → 0.45，边框从 0.6 → 0.85（原色太淡在深色背景下不可见）
- 仅支持 `.zip`/`.7z` 内含标准 `minecraft:geometry` 的模型

### 📦 YSM 二进制格式识别

- `.ysm` 是 YSM 专有二进制编码格式，非普通 zip，不包含明文的 `minecraft:geometry`
- 预览区显示友好提示："YSM 二进制格式 · 需解析器解码后方可查看骨骼结构"
- 不再尝试无效解析，避免误判

### 🐛 Bug 修复

- **Go 编译修复**：`parseBedrockFromZip`/`parseBedrockGeometry` 中的 debug println 导致编译警告
- **前端加载安全**：`_loadModel2D` 先插占位符再异步加载，避免因 JS 截断导致 UI 丢失
- **`.ysm` 跳过修复**：回退之前错误添加的 `.ysm` → zip 解析逻辑（YSM 是二进制非 zip）

### 🔧 调试增强

- 预览区即时显示 `🏗️ 模型结构（读取中...）` 占位符
- 失败时显示具体错误信息而非静默失败
- 独立测试页 `prototypes/model2d-test.html`，可在浏览器中直接验证 2D 渲染

## 🧪 技术说明

- YSM 模型格式需专用解析器（[YSMParser](https://github.com/YSMParser) WASM/C++）解码后才可提取骨骼数据
- 目前仅支持标准 Bedrock geometry 格式（`.zip`/`.7z` 内含 `minecraft:geometry`）
- `.ysm` 支持需后续集成 YSMParser
