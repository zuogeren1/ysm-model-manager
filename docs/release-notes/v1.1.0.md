# v1.1.0 — MC 目录智能检测 + CSS 工程化

> 发布日期: 2026-06-08

## 🎯 核心改动

### 🧭 MC 目录智能检测

- **内容级检测**：检查 `versions/`、`assets/`、`launcher_profiles.json` 特征子目录/文件，不再仅靠路径名判断
- **自动修正**：用户误选 PCL2 父目录（如 `D:/PCL2/`）时，自动检测 `D:/PCL2/.minecraft/` 并跳转
- **扩展扫描**：覆盖 10+ 常见位置 — PCL2/PCL/MC/Minecraft/Games 各盘符 + `%APPDATA%`
- **首次启动自动检测**：未设置时自动取第一个有效路径作为默认
- **错误提示**：选错目录时弹 toast 说明原因
- 设置页「自动搜索」按钮改用新 API，不再解析显示字符串
