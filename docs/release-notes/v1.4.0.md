# v1.4.0 — 纹理筛选、仓库健康度、今日推荐

> 发布日期: 2026-06-10

## ✨ 新功能

### 🔍 纹理尺寸筛选现在可用了

仓库顶部的「📐 纹理 ▾」下拉菜单和「🔍 筛选」面板现在能正常过滤文件树了。按纹理尺寸、骨骼数量、立方体数量筛选后，树形列表只显示匹配的文件。

### 👴 诊断页新增「仓库元老」彩蛋

诊断页左侧新增一个按钮，打开后可以看到：

- **仓库健康度** 🟢 — 综合你的仓库状态给出评分和评语
- **仓库元老** 👴 — 找出你最早拥有的模型，显示它陪伴了你多久
- **今日推荐** 🎲 — 每次打开随机推荐一个模型

### 🖼️ 模型详情显示纹理尺寸

现在打开模型详情，除了骨骼数、贴图数外，还会显示纹理的实际分辨率（如 2048×2048 px）。

### 🔗 去重功能增强

模型去重时每组文件下方会显示所在的文件夹路径，方便判断该保留哪个。

## 🔧 改进

- 仓库搜索和筛选功能整体更稳定、响应更快
- 设置页保存路径时，选 `C:\PCL2` 会自动补全为 `C:\PCL2\.minecraft`
- 对话框、重命名弹窗、批量重命名等界面样式全部恢复正常
- 配置文件被删除后重启会自动重建并尝试检测游戏路径

## 🐞 修复

- 修复配置文件因校验失败而无法创建的问题
- 修复全局 CSS 加载缺失导致大量 UI 样式丢失
- 修复选择游戏目录后路径未自动修正的问题
