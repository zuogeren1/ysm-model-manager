## 1️⃣ 下载队列（workshop）

### 前置条件

- 打开创意工坊 → 点击一个 GitHub 创作者 → 📦 浏览
- 确保仓库里有 index.json（否则"无索引"是正常行为）

### 测试项

| #    | 测试             | 操作                                         | 预期                                            |
| ---- | ---------------- | -------------------------------------------- | ----------------------------------------------- |
| 1.1  | **单击下载**     | 点击模型行任意位置或 ⬇️ 按钮                 | 出现进度条，显示文件名 + 百分比，完成后刷新列表 |
| 1.2  | **多选下载**     | 勾选多个复选框 → 点击「⬇️ 下载选中 (N)」     | 串行下载，进度条依次切换                        |
| 1.3  | **全选下载**     | 点击「☐ 全选」→「⬇️ 下载选中 (N)」           | 所有缺失模型入队                                |
| 1.4  | **进度条**       | 下载中观察                                   | 显示百分比 + 进度条，小文件看 99%→100%动画      |
| 1.5  | **取消下载**     | 下载中点击 ✕ 取消按钮                        | 队列中止，按钮恢复                              |
| 1.6  | **下载全部缺失** | 点击「⬇️ 下载全部缺失」                      | 弹出确认框，确认后串行下载                      |
| 1.7  | **错误处理**     | 下载中途断网                                 | 进度条变红 ❌，队列结束后显示错误摘要           |
| 1.8  | **已有文件**     | 下载完成后再次浏览同一仓库                   | 已下载的模型显示 ✅ 已有，不可点击              |
| 1.9  | **镜像源**       | 设置页切换镜像源（直连/jsDelivr/GitHub API） | 下载仍正常，进度条仍工作                        |
| 1.10 | **小文件动画**   | 下载 <100KB 的文件                           | ⏳ → 99%（150ms）→ 100%                         |
